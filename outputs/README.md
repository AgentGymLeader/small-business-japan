# Small Business Japan Outputs

Each output has the same reviewable shape:

- `output.md` defines what the workflow may produce.
- `research/2026-05-14.md` records the Japan-specific research boundary checked
  for that output.
- `research/latest.md` points maintainers to the current dossier.

The v0 output policy is deliberately narrow: review memo, owner-review draft,
checklist, and professional confirmation packet only. Extracted-field
candidates, missing-data lists, and risk/escalation notes may appear only as
sections inside those containers. The workflows do not send, file, submit, pay,
refund, sign, publish, update systems, create tool-specific import CSVs, or make
final professional determinations.

Refusal / hold triggers include requests to submit, file, register, represent,
sign, send, pay, refund, publish, finalize, certify compliance, prepare tax
returns, create import CSVs, or make final legal/tax/labor/accounting decisions.
