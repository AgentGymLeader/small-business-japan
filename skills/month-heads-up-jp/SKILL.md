# Month Heads-Up JP

## Quick start

来月（または指定した月）の資金繰りを先読みし、
不足リスク・重要な期限・必要なアクションを整理する。

```
オーナー: 「来月の資金繰りを確認して」
→ 来月の入金予定集計
→ 来月の支出・納付期限集計
→ 資金不足リスクの評価
→ 要対応アクションリストの作成
```

## Workflow

1. **対象月を確認する。** デフォルトは翌月。「再来月」「3ヶ月後」などの指定があればそれに従う。

2. **入金予定を集計する。** freee または マネーフォワードクラウドから対象月の入金予定を取得する：
   - 発行済みで未収の請求書（支払期限が対象月内）
   - 定期契約・サブスクリプション収入（前月実績から推定）

3. **支出予定を集計する。** 対象月内の支払い予定を一覧化する：
   - 仕入・外注費の未払い
   - 家賃・リース等の固定費
   - 人件費（給与支払日）
   - 社会保険料・源泉所得税・住民税の納付期限

4. **重要な期限をカレンダー表示する。** 対象月内の重要な期限を日付順に並べる：
   - 請求書の支払期限（大口のもの）
   - 社会保険料の納付期限（末日）
   - 源泉所得税の納付期限（10日）
   - 住民税の納付期限（10日）
   - 消費税の申告・納付期限（該当月の場合）
   - 決算・確定申告の期限（該当月の場合）

5. **資金不足リスクを評価する。** 日別キャッシュフローシミュレーションを行い、
   残高がマイナスになる日・期間があれば警告する。
   「{日付}頃に{金額}円の不足リスクがあります」と具体的に表示する。

6. **アクションリストを作成する。** 今週中に対応すべき事項を優先度順にまとめる：
   - 督促が必要な未収入金
   - 資金が不足する場合の対応策（融資相談、支払い交渉等）
   - 税理士・社労士への確認事項

## Approval gates

- **外部への送信・書き込みは行わない。** 分析・報告のみ。
- **資金移動・融資申請の実行は行わない。**
- **税金計算の最終確認は税理士に依頼するよう促す。**
- オーナーの明示的な承認なしに、資金移動・外部送信・支払い指示の実行は行わない。
