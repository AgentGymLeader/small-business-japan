# Invoice Chase JP

## Quick start

期限超過の請求書を特定し、取引先ごとに督促メールの下書きを作成する。
送信前に必ずオーナーの確認を得る。

```
オーナー: 「未入金の請求書を確認して督促メールを書いて」
→ 期限超過請求書の一覧取得
→ 取引先別に滞納期間・金額を整理
→ 督促メールの下書き作成
→ オーナーが確認・承認後に送信
```

## Workflow

1. **期限超過請求書を取得する。** freee または マネーフォワードクラウドから未収の請求書を取得する。支払期限を過ぎているものをフィルタリングして一覧化する。

2. **優先度を設定する。** 以下の基準で督促の優先度を決める：
   - **最優先**: 30日以上超過、かつ金額が大きい
   - **優先**: 7〜30日超過
   - **通知**: 1〜7日超過（支払期限翌日〜）

3. **取引先情報を確認する。** 各取引先の過去の支払い履歴を確認し、以下を把握する：
   - 初めて遅れている取引先か
   - 過去にも遅延があった取引先か
   - 現在進行中の商談・取引があるか

4. **督促メールの下書きを作成する。** 滞納期間と関係性に応じてトーンを調整する：
   - 初回遅延・良好な関係 → 丁寧な確認メール
   - 2回目以降 → やや明確な督促
   - 長期滞納 → 正式な督促（法的措置の可能性を示唆）

   メールには必ず含める：請求書番号、金額、支払期限、振込先情報。

5. **オーナーに確認を求める。** 下書きを提示し、送信前に承認を得る。
   修正依頼があれば対応し、再度確認する。

6. **インボイス制度の確認フラグを立てる。** 以下は「確認が必要」としてフラグするのみ。確定判断はしない：
   - 適格請求書発行事業者番号（T + 13桁）の形式確認（公的登録確認は未実施・タイムスタンプなしと明記）
   - 軽減税率（8%）の対象品目への軽減税率マーカーの有無
   - 税率ごとの税抜き小計・消費税額の記載
   - 買い手名の記載（簡易インボイスの適用可能性がある場合は別途確認）
   - 免税事業者からの請求書: 経過措置（2023〜2029年）が適用される可能性があるため、「要確認」とし欠陥と断定しない

## Approval gates

- **オーナーの明示的な承認なしにメールを送信しない。** 下書きのみ提示する。
- **請求書のステータスをシステム上で変更しない。** 読み取り専用。
- **法的措置に関する確定的な表現を使わない。** 「弁護士にご相談ください」と促すにとどめる。
- **インボイスが「適格請求書」として有効かを確定的に判断しない。** フラグと確認事項の提示のみ。
- **仕入税額控除の可否を確定的に判断しない。** 税理士に確認するよう促す。
- **免税事業者の請求書を欠陥と断定しない。** 経過措置の適用可能性があり、税理士確認が必要。
- **督促メールに「インボイスの発行義務があります」等の断定的な記載を含めない。**
