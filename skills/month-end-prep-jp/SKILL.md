# Month End Prep JP

## Quick start

月次締め前に必要な確認事項を自動チェックし、
漏れ・未処理項目を一覧化して税理士との確認ミーティングに備える。

```
オーナー: 「月次締め準備をして」
→ 未分類・未処理取引の検出
→ 証憑（領収書）の抜けチェック
→ 未収・未払いの確認
→ 月次締めチェックリストの作成
```

## Workflow

1. **未分類取引を検出する。** freee または マネーフォワードクラウドで勘定科目が未設定または「未分類」の取引を抽出する。取引数と金額の合計を表示する。

2. **証憑の抜けを確認する。** 一定金額（デフォルト: 1万円）以上の支出について、領収書・証憑が添付されているか確認する。添付されていないものをリストアップする。

3. **未収入金を確認する。** 当月の請求書で未入金のものを一覧化する。次の区分で整理する：
   - 支払期限内（予定通り）
   - 支払期限超過（督促が必要）

4. **未払い費用を確認する。** 当月中に支払い予定だったが未払いのものを一覧化する。

5. **固定費の計上漏れを確認する。** 毎月発生するはずの費用（家賃、リース料、保険料等）が当月に計上されているか確認する。前月比較で抜けを検出する。

6. **インボイス対応を確認する。** 当月の仕入・経費について：
   - 適格請求書（インボイス）として受け取った請求書の枚数・金額
   - 仕入税額控除の対象となるもの / ならないものの内訳
   - 電子帳簿保存法（電帳法）対応の確認: 電子取引データの保存要件（検索機能・タイムスタンプ等）が満たされているか「確認が必要」としてフラグする。保存要件の充足を確定的に判断しない。

7. **月次締めチェックリストを出力する。** 以下の形式でまとめる：
   ```
   月次締めチェックリスト — {年月}

   ✅ 完了している項目
   ⚠️ 要対応の項目
     - 未分類取引: {件数}件（合計{金額}円）
     - 証憑なし支出: {件数}件（合計{金額}円）
     - 期限超過未収入金: {件数}件（合計{金額}円）

   📋 税理士への確認事項
     - {確認が必要な事項のリスト}
   ```

## Approval gates

- **帳簿の自動修正・仕訳の確定は行わない。** チェックと報告のみ。
- **税務判断が必要な項目は税理士への相談を促す。**
- **外部への送信は行わない。**
- **仕訳・勘定科目の承認・確定は行わない。** 「候補」「要確認」として提示する。
- **電子帳簿保存法の保存要件充足の最終確認は税理士に依頼するよう促す。**
- **「月次締め完了」「申告データ準備完了」とは言わない。**
