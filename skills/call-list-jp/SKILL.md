# Call List JP

## Quick start

`lead-triage-jp` のコール特化版。今日電話すべき優先顧客・見込み客リストを作成し、
各社へのアプローチポイントを提示する。

```
オーナー: 「今日電話するリストを作って」
→ kintone / スプレッドシートからデータ取得
→ コール優先度のスコアリング
→ 今日のコールリスト（最大10社）を作成
→ 各社へのアプローチポイントを提示
```

## Workflow

1. **データを取得する。** kintone またはスプレッドシート（顧客管理）から以下をフィルタリングする：
   - フォローアップ日が今日以前になっているレコード
   - 商談フェーズが「商談中」「提案中」「クロージング」のもの
   - 過去30日以内に接触のない見込み客

2. **コール優先度をスコアリングする。** `lead-triage-jp` と同じ4軸を使用する：
   - エンゲージメント（最近の返信・資料開封等）
   - 適合性（ターゲット業種・規模との一致度）
   - 緊急度（期限・決裁タイミング・キーワード）
   - 最終接触からの経過日数

3. **今日のコールリストを作成する。** スコア上位10社（または全件がそれ以下なら全件）をリストアップする。各社について：
   - 会社名・担当者名
   - 最後に何を話したか（1文）
   - 今日の話題・アプローチポイント（1〜2文）
   - 推奨するアプローチトーン（フォローアップ / 提案 / クロージング）

4. **トーキングポイントを提示する。** 各社への具体的な会話の糸口を1〜2文で提案する。
   Stripe や freee のデータがある場合は、最新の購入・取引情報も参照する。

## Approval gates

- **メールの送信・カレンダーへの登録は行わない。**
- **個人情報（電話番号・メールアドレス）を出力の目立つ位置に表示しない。**
- **顧客データの更新・書き込みは行わない。** 読み取り専用。
- **オーナーの明示的な承認なしにkintone・スプレッドシートのレコードを変更しない。**
