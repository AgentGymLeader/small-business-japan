# Tax Season Organizer JP

## Quick start

税理士との確認ミーティングに向けて、会計データの整理・不明点の洗い出しを行い、
確認パケット（メモ形式）を作成する。申告書の作成・提出は行わない。

```
オーナー: 「税理士確認パケットを作って」
→ 会計データの整理状況確認
→ 未分類・未処理項目の検出
→ 税理士への確認事項リスト作成
→ 確認パケット（メモ）の作成
```

## Workflow

1. **申告種別を確認する。** 個人事業主（確定申告）か法人（法人税申告）かを確認する。
   わからない場合は質問する。

2. **対象期間を確認する。** 個人事業主の場合は前年1月1日〜12月31日、
   法人の場合は事業年度を確認する。

3. **会計データの整理状況を確認する。** freee または マネーフォワードクラウドで：
   - 未分類取引の件数・金額
   - 証憑（領収書）未添付の取引
   - 前期残高との不一致
   をチェックし、優先度の高いものから整理を促す。

4. **インボイス・消費税の確認を行う。**
   - 適格請求書発行事業者の場合: 発行済みインボイスの登録番号（T + 13桁）の確認
   - 課税事業者の場合: 仕入税額控除の対象となる取引の集計
   - 簡易課税・本則課税の選択届出の状況確認

5. **税理士への確認事項をリストアップする。** 以下の項目について確認事項を整理する：
   - 減価償却資産の購入・廃棄
   - 役員報酬の変更（法人の場合）
   - 交際費・会議費の区分
   - 家事按分が必要な費用（自宅兼事務所等）
   - 生命保険料控除等の各種控除
   - 青色申告特別控除の適用条件

6. **確認パケット（メモ）を作成する。** 以下の形式でまとめる：

   ```
   税理士確認パケット — {事業年度}

   【基本情報】
   - 事業形態: 個人事業主 / 法人
   - 対象期間: {期間}
   - 申告期限: {日付}

   【データ整理状況】
   - 未分類取引: {件数}件（合計{金額}円）
   - 証憑未添付: {件数}件

   【確認事項リスト】
   1. {確認事項}
   2. {確認事項}
   ...

   【インボイス・消費税】
   - {確認事項}

   ※ このパケットは税理士との確認ミーティング用の整理メモです。
     申告書の内容や税務判断については税理士にご確認ください。
   ```

## Approval gates

- **申告書の作成・送信・提出は行わない。** 整理とメモの作成のみ。
- **税務判断（控除の適否・申告方法の選択等）は税理士に相談するよう促す。**
- **「このまま申告できる」「税務上問題ありません」とは言わない。**
- **外部への送信・電子申告の実行は行わない。**
- オーナーの明示的な承認なしに、電子申告システムへのデータ送信・申告書の提出は行わない。
- **控除・税額・税区分の最終決定は行わない。** すべて「確認事項」として提示する。
- **電子帳簿保存法（電帳法）の保存要件充足を確定的に判断しない。**
- **申告ソフトへの取り込みデータ・税務申告書の作成は行わない。**
- **「この申告で問題ありません」「控除が適用されます」とは言わない。**
- 税理士法違反（税理士でない者による税務相談の代行）に該当する表現を使わない。（国税庁「No.9204 にせ税理士にご注意」準拠）
