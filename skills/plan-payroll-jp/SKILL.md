# Plan Payroll JP

## Quick start

今月・来月の給与、社会保険料、源泉所得税、住民税の支払い予定を整理し、
資金繰りとの照合を行う。社労士・税理士への確認事項もリストアップする。

```
オーナー: 「支払準備をして」
→ 給与支払日・金額の確認
→ 社会保険料の納付期限確認
→ 源泉所得税・住民税の納付期限確認
→ 現金残高との照合
→ 支払計画レポートの作成
```

## Workflow

1. **給与支払いを確認する。** freee 人事労務 または マネーフォワードクラウド給与から当月の給与データを取得する。接続されていない場合は、オーナーに給与総額と支払日を入力してもらう。

2. **社会保険料の納付を確認する。** 以下の確認を行う：
   - 健康保険料・厚生年金保険料の納付期限（毎月末日）
   - 雇用保険料の申告・納付期限
   - 労災保険の確認

3. **源泉所得税の納付を確認する。** 
   - 毎月納付の場合: 翌月10日が期限
   - 納期の特例の場合: 1月20日・7月10日が期限
   どちらが適用されているか確認し、直近の納付期限を表示する。

4. **住民税（特別徴収）を確認する。**
   - 毎月10日が納付期限
   - 対象の従業員数と合計額を確認する

5. **資金繰りと照合する。** 現在の口座残高と以下の支払い合計を比較する：
   - 給与支払い総額
   - 社会保険料（会社負担分 + 従業員負担分）
   - 源泉所得税
   - 住民税

   残高不足のリスクがある場合は警告を表示する。

6. **支払計画レポートを作成する。** 以下の形式でまとめる：
   ```
   支払計画 — {年月}

   給与支払い
   - 支払日: {日付}
   - 総支給額: {金額}円
   - 手取り総額: {金額}円

   社会保険・税金納付
   - 社会保険料: {金額}円（期限: {日付}）
   - 源泉所得税: {金額}円（期限: {日付}）
   - 住民税: {金額}円（期限: {日付}）

   現在残高: {金額}円
   支払後残高（予測）: {金額}円
   ```

7. **社労士・税理士への確認事項をリストアップする。** 以下があれば提示する：
   - 雇用形態の変更（パート→正社員等）による社保変更
   - 産休・育休中の社保免除の確認
   - 年末調整に向けた準備事項（10月以降）

## Approval gates

- **給与の振込・支払い実行は行わない。** 計画と確認のみ。
- **社会保険・税金の申告・納付は行わない。**
- **社労士・税理士の確認が必要な事項は明示して促す。**
- **従業員の個人情報（氏名・給与明細）を出力内に含める場合はオーナーの明示的な承認なしに出力しない。**
