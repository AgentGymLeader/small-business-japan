# Cash Flow Snapshot JP

## Quick start

freee / マネーフォワードクラウドの残高、未払い請求書、今後30日の支出予定を照合し、
資金繰りの見通しをまとめる。外部への送信・書き込みは行わない。

```
オーナー: 「今月の資金繰りを見せて」
→ 現金残高の取得
→ 入金予定（未収請求書）の集計
→ 支出予定（未払い費用・固定費）の集計
→ 30日間の資金繰り見通しを作成
→ 懸念点がある場合は強調表示
```

## Workflow

1. **現金残高を取得する。** freee または マネーフォワードクラウドの銀行・現金口座残高を取得する。接続されていない場合は止まって報告する。

2. **入金予定を集計する。** 次の30日以内に入金予定の未収請求書を取得し、合計する。freee の場合は売掛金レポート、マネーフォワードクラウドの場合は請求書一覧から取得する。

3. **支出予定を集計する。** 次の30日以内に支払い予定の費用を集計する：
   - 未払い請求書・買掛金
   - 家賃・リース等の固定費（前回実績から推定）
   - 社会保険料・税金の納付期限
   - その他定期支払い

4. **資金繰り見通しを作成する。** 以下の形式で出力する：
   - **現在の残高**: 円
   - **30日後の予測残高**: 円（入金予定 − 支出予定）
   - **最低残高予測日**: いつどの程度まで下がるか
   - **注意ポイント**: 残高がマイナスになるリスクがある期間

5. **懸念事項を強調表示する。** 以下の場合は目立つよう表示する：
   - 予測残高が現在残高の50%を下回る
   - 30日以内に残高不足リスクがある
   - 大口の入金と支払いが重なっている

## Approval gates

- **外部への送信・書き込みは行わない。** 読み取り専用の分析のみ。
- **数値の確定判断は税理士に相談するよう促す。**
- **資金移動・振込・支払いの実行は行わない。**
