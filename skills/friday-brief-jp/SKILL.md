# Friday Brief JP

## Quick start

今週の成果と課題を振り返り、来週へのアクションを整理した
週次ふりかえりレポートを作成する。

```
オーナー: 「金曜のふりかえりをして」
→ 今週完了したこと・未完了のことを確認
→ 入金・支払い実績の確認
→ 来週に持ち越すアクションの整理
→ 振り返りレポートの作成
```

## Workflow

1. **今週の入金・支払い実績を確認する。** freee / マネーフォワードクラウドから今週の実績を取得する：
   - 今週の売上・入金合計
   - 今週の支払合計（消費税込み金額の確認）
   - 期限内に完了した支払い / 未払いのまま持ち越し

2. **未完了の督促を確認する。** 今週督促すべきだった未収入金のうち、まだ解決していないものをリストアップする。

3. **フォローアップ済み / 未済を確認する。** 今週中に予定されていたフォローアップのうち：
   - 完了済み
   - 未済（来週に持ち越し）

4. **来週の優先事項を整理する。** 来週のブリーフとなるアクションを作成する。

5. **振り返りレポートを作成する。** 以下の形式でまとめる：

   ```
   金曜ふりかえり — {日付}の週

   📊 今週の実績
   ・売上・入金: {金額}円
   ・支払合計: {金額}円
   ・差引: +/- {金額}円

   ✅ 今週完了したこと
   ・{完了事項}
   ・{完了事項}

   ⚠️ 未完了・持ち越し
   ・{未完了事項} — 来週{日付}までに対応
   ・{未完了事項}

   📅 来週の優先事項
   1. {アクション} — 期限: {日付}
   2. {アクション}
   3. {アクション}
   ```

## Approval gates

- **オーナーの明示的な承認なしに、外部への送信・書き込みは行わない。** ふりかえりレポート作成のみ。
- **帳簿の確定・締めはオーナーが税理士と確認してから行う。**
- **消費税・インボイス制度の仕訳・申告に関する実行はオーナーが税理士と確認してから行う。**
