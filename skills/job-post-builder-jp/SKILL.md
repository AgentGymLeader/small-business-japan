# Job Post Builder JP

## Quick start

採用したいポジションの情報をヒアリングし、
Indeed・求人ボックス・自社採用ページ向けの求人票を下書きする。
求人票の投稿・掲載は行わない。

```
オーナー: 「エンジニアの求人票を作りたい」
→ ポジション・業務内容のヒアリング
→ 労働条件（給与・勤務時間・福利厚生等）の確認
→ 求人票の下書き作成
→ 労働基準法の必須記載事項チェック
→ オーナー確認・修正
```

## Workflow

1. **基本情報をヒアリングする。** 以下を1つずつ確認する（一度に全部聞かない）：
   - 採用したいポジション・職種
   - 業務内容・主な仕事（箇条書きで3〜5点）
   - 雇用形態（正社員・パート・業務委託等）
   - 勤務地・テレワーク可否

2. **労働条件を確認する。** 以下を確認する：
   - 給与・報酬（固定か変動か、月給/時給）
   - 所定労働時間・勤務曜日（労働基準法に基づく週40時間・1日8時間の原則）
   - 試用期間（あれば）
   - 社会保険の加入有無
   - 休日・有給休暇（労働基準法上の年次有給休暇の付与義務を確認）

3. **求める人物像を確認する。** 以下を確認する：
   - 必須スキル・経験
   - 歓迎スキル・経験
   - 求める人柄・マインドセット

4. **求人票を下書きする。** 以下の構成で作成する：
   ```
   【求人票】{ポジション名}

   ■ お仕事内容
   {業務内容}

   ■ 応募資格
   必須: {必須スキル・経験}
   歓迎: {歓迎スキル・経験}

   ■ 雇用形態
   {雇用形態}

   ■ 給与・報酬
   {給与}
   ※試用期間: {期間と条件}

   ■ 勤務時間
   {所定労働時間}

   ■ 勤務地
   {勤務地}

   ■ 休日・休暇
   {休日・有給等}

   ■ 社会保険
   {加入状況}

   ■ 福利厚生
   {福利厚生}
   ```

5. **労働基準法の必須記載事項を確認する。** 以下が網羅されているか確認し、不足があれば追記を促す：
   - 労働契約期間（定めなし or 期間）
   - 就業場所・業務内容
   - 所定労働時間・休憩時間
   - 休日・休暇
   - 賃金（決定・計算・支払い方法、時期）
   - 退職・解雇条件
   - 就業規則が存在する場合はその周知方法
   - 2024年4月施行の労働条件明示ルール（職業安定法施行規則改正）の対応:
     - 有期雇用の場合: 更新上限（上限回数・期間）の明示
     - 固定残業代がある場合: 固定残業時間・金額の明示
     - 就業場所・業務の変更範囲の明示
     - これらが記載されていない場合は追記を促す
   - 公正採用選考の観点: 採用基準・選考方法に本人の適性・能力以外の要素（家族構成・出身地等）が含まれていないか確認

6. **禁止表現を確認する。** 以下の表現が含まれていれば削除を促す：
   - 年齢制限（「35歳以下」等）— 雇用対策法により原則禁止
   - 性別指定（「男性優遇」等）— 男女雇用機会均等法により禁止
   - 国籍・出身地による差別的表現

## Approval gates

- **オーナーの明示的な承認なしに、求人の投稿・掲載（Indeed・求人ボックス等）は行わない。** 下書きのみ。
- **社労士確認が必要な労働条件（残業代の計算・みなし労働時間・裁量労働制等）があれば指摘する。**
- **雇用契約書の作成・署名は行わない。** 雇用契約書の締結は社労士または弁護士への相談を促す。
- **就業規則の策定・変更は社労士への相談を促す。**
- **「この求人票は法的に問題ありません」「差別リスクは排除されています」とは言わない。**
- **年齢・性別制限の例外適用（雇用対策法・男女雇用機会均等法）の判断は厚生労働省・専門家に確認するよう促す。**
- **応募者の個人情報は採用目的のみ（個人情報保護法第18条）であることをオーナーに確認するよう促す。**
- 社労士・弁護士への確認が必要な労働条件（みなし労働時間・フレックスタイム等）が含まれる場合は必ず指摘する。
