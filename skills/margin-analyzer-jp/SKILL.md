# Margin Analyzer JP

## Quick start

売上と原価のデータを取得し、商品・サービス別の粗利・利益率を計算して
改善ポイントを提示する。

```
オーナー: 「どの商品が一番儲かってるか確認して」
→ 売上データの取得（期間指定可）
→ 原価・費用データの取得
→ 商品・サービス別の利益率計算
→ 改善提案のレポート作成
```

## Workflow

1. **分析期間を確認する。** デフォルトは過去90日。オーナーが期間を指定した場合はそれに従う。

2. **売上データを取得する。** freee または マネーフォワードクラウドから商品・サービス別の売上を取得する。
   - 可能であれば品目・サービス種別ごとに分類する
   - 消費税（8%/10%）の区分を確認する

3. **原価・費用データを取得する。** 売上に対応する原価・直接費用を取得する：
   - 仕入原価
   - 外注費
   - 材料費・消耗品費

4. **利益率を計算する。** 各商品・サービスについて算出する：
   - **粗利** = 売上 − 原価
   - **粗利率** = 粗利 ÷ 売上 × 100
   - 全体の粗利率と比較して高い・低いを評価する

5. **改善提案を作成する。** 以下のパターンで提案する：
   - 粗利率が全体平均より20%以上低い商品 → 「価格見直しまたは原価削減を検討」
   - 売上高は低いが粗利率が高い商品 → 「プロモーション強化を検討」
   - 原価が売上の80%を超える商品 → 「採算性の再検討を推奨」

6. **税理士へのエスカレーション提案。** 利益率の改善が複数期間にわたって必要な場合、
   「税理士に相談して節税・コスト構造の見直しを検討することをお勧めします」と提案する。

## Approval gates

- **外部への送信・書き込みは行わない。** 分析・レポートのみ。
- **価格変更・システム更新は行わない。** 提案のみ。
- **会計上の確定的な判断は税理士に相談するよう促す。**
