# Monday Brief JP

## Quick start

今週1週間の優先事項・期限・アクションを整理して、
週のはじめにすぐ動けるブリーフを作成する。

```
オーナー: 「月曜ブリーフを作って」
→ 今週の支払期限確認
→ 未収入金・フォローアップ確認
→ 今週のスケジュール確認
→ 優先アクションリストの作成
```

## Workflow

1. **今週の期限を確認する。** 以下を今週の日付でフィルタリングする：
   - freee / マネーフォワードクラウドの請求書支払期限
   - 社会保険料・源泉所得税・住民税の納付期限
   - 消費税（インボイス制度対応含む）の申告・納付期限（該当週の場合）

2. **未収入金を確認する。** 期限超過の未収請求書をリストアップし、今週中に督促すべきものを特定する。

3. **フォローアップ予定を確認する。** kintone / スプレッドシートでフォローアップ日が今週以内に設定されているレコードを確認する。

4. **今週の重要イベントを確認する。** 試算表の締め、資金繰りのピーク日、税務申告等を一覧化する。

5. **月曜ブリーフを作成する。** 以下の形式でまとめる：

   ```
   月曜ブリーフ — {日付}の週

   ✅ 今週の優先事項
   1. {アクション} — 期限: {日付}
   2. {アクション} — 期限: {日付}
   3. {アクション}

   💰 今週の支払・入金
   ・入金予定: {金額}円（{件数}件）
   ・支払予定: {金額}円（{件数}件）
   ・期限超過の督促が必要: {件数}件

   📅 今週の期限
   ・{日付}: {期限内容}
   ・{日付}: {期限内容}

   📞 フォローアップ予定
   ・{会社名} — {用件}
   ```

## Approval gates

- **オーナーの明示的な承認なしに、外部への送信・書き込みは行わない。** ブリーフ作成のみ。
- **メールの送信・カレンダーへの登録はオーナーが行う。**
- **消費税・インボイス制度に関する申告・納付の実行はオーナーが税理士と確認してから行う。**
