# Close Month JP

## Quick start

月次締めの全工程を順番に実行し、月次レポートを完成させる。
各ステップでオーナーの確認を得ながら進める。

## Workflow

### Step 1 — 月次チェック（month-end-prep-jp）

`month-end-prep-jp` スキルを実行する：
1. 未分類取引・証憑抜けの検出
2. 未収・未払いの最終確認
3. 固定費の計上漏れチェック
4. インボイス制度下での適格請求書・消費税の確認

Step 1 の結果を提示する。重要な未処理項目がある場合は、
「この項目を解決してから月次締めを進めてください」と促してオーナーの確認を得る。

### Step 2 — 資金繰りサマリー（cash-flow-snapshot-jp）

`cash-flow-snapshot-jp` スキルを実行する：
1. 当月の現金残高の推移
2. 入金・支出の実績
3. 来月の資金繰り見通し

### Step 3 — 月次レポートの統合

両スキルの結果を統合してサマリーを作成する：

```
月次レポート — {年月}

【データ整理状況】
{month-end-prepの結果サマリー}

【当月実績】
・売上: {金額}円
・費用: {金額}円
・粗利: {金額}円（{%}）
・期末現金残高: {金額}円

【来月の見通し】
{cash-flow-snapshotの資金繰り見通し}

【税理士への確認事項】
{確認事項リスト}
```

### Step 4 — 税理士へのエスカレーション提案

確認事項がある場合、以下を提案する：
「税理士に以下の事項をご確認ください：{確認事項リスト}」

## Approval gates

各ステップのApproval gatesに従う。
- **帳簿の確定・締めはオーナーが税理士と確認してから行う。**
- **申告書の作成・提出は行わない。**
- **各ステップの間でオーナーの確認を得る。**
- **オーナーの明示的な承認なしにfreee・マネーフォワードクラウドの帳簿データを修正しない。**
