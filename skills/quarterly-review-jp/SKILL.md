# Quarterly Review JP

## Quick start

直近の四半期（または指定した期間）の事業実績を集計し、
成果・課題・次の四半期へのアクションプランをまとめる。

```
オーナー: 「四半期レビューをして」
→ 売上・収益の実績集計
→ 目標達成率の評価
→ 課題・強み・機会の分析
→ 次の四半期の優先事項の提案
```

## Workflow

1. **対象期間を確認する。** デフォルトは直前の四半期（1〜3月、4〜6月、7〜9月、10〜12月）。指定があればそれに従う。

2. **売上・収益の実績を集計する。** freee または マネーフォワードクラウドから対象期間のデータを取得する：
   - 総売上・純売上
   - 総費用
   - 粗利・粗利率
   - 前四半期比・前年同期比

3. **商品・サービス別の実績を確認する。** 売上の内訳を商品・サービス別に集計し、以下を特定する：
   - 最も貢献した商品・サービス（上位3位）
   - 想定より低迷した商品・サービス

4. **資金繰りの実績を確認する。** 四半期を通じた：
   - 現金残高の推移（期首 vs 期末）
   - 最低残高と発生時期
   - 未収入金の残高

5. **目標との対比を行う（目標が設定されている場合）。** オーナーに四半期の目標を確認し、達成率を計算する。目標が設定されていない場合はスキップ。

6. **KSF（重要成功要因）を分析する。** 以下の視点で強みと課題を整理する：
   - 売上：伸びた要因 / 伸び悩んだ要因
   - コスト：想定内 / 超過した費用カテゴリ
   - 顧客：新規獲得数 / 既存顧客の継続率（データがある場合）

7. **次の四半期の優先事項を提案する。** 分析結果に基づき、来期に重点的に取り組むべき3〜5項目を提案する。

8. **四半期レビューレポートを作成する。** 以下の形式でまとめる：

   ```
   四半期レビュー — Q{N} {年}

   【実績サマリー】
   ・総売上: {金額}円（前四半期比: {+/-}%）
   ・粗利率: {%}（前四半期: {%}）
   ・期末現金残高: {金額}円（期首比: {+/-}円）

   【売れ筋 TOP3】
   1. {商品/サービス}: {売上}円
   2. {商品/サービス}: {売上}円
   3. {商品/サービス}: {売上}円

   【課題・改善点】
   ・{課題}
   ・{課題}

   【次の四半期の優先事項】
   1. {優先事項}
   2. {優先事項}
   3. {優先事項}

   【税理士・社労士への確認事項】
   ・{確認事項}
   ```

## Approval gates

- **外部への送信・書き込みは行わない。** レポート作成のみ。
- **財務予測・将来の収益保証は行わない。**
- **税務に関わる事項は税理士に確認するよう促す。**
- **オーナーの明示的な承認なしに帳簿データの修正・更新を行わない。**
