# Contract Review JP

## Quick start

契約書テキストを貼り付けると、重要条項・リスクポイント・
確認すべき論点を整理したメモを作成する。
法的アドバイスではなく、弁護士確認前の整理ツール。

```
オーナー: 「この契約書の論点を確認して」
→ 契約書テキストの読み取り
→ 重要条項・リスクの抽出
→ 確認事項リストの作成
→ 弁護士・行政書士への確認推奨事項の提示
```

## Workflow

1. **契約書を受け取る。** テキストとして貼り付けてもらう。PDF の場合は、コピーできるテキスト部分を貼り付けてもらう。

2. **基本情報を確認する。** 以下を特定する：
   - 契約の種類（業務委託・売買・賃貸借・秘密保持等）
   - 当事者（依頼人側・相手方）
   - 契約期間・更新条件
   - 準拠法・管轄裁判所

3. **重要条項を抽出する。** 以下のカテゴリごとに整理する：
   - **報酬・支払条件**: 金額、支払期限、遅延損害金
   - **業務範囲**: 委託業務の定義、追加作業の扱い
   - **秘密保持**: 期間、対象情報、罰則
   - **知的財産権**: 成果物の帰属、使用許諾の範囲
   - **解除・解約**: 解除条件、解約通知期間、違約金
   - **損害賠償・免責**: 責任の上限、免責事由
   - **自動更新**: 更新の条件、解約意思表示の期限

4. **リスクポイントを特定する。** 以下のパターンがあれば注意を促す：
   - 一方的に不利な解除条件
   - 損害賠償の上限が設定されていない条項
   - 曖昧な業務範囲の定義
   - 過度に長い秘密保持期間（5年超等）
   - 知的財産権が相手方に帰属する条項
   - 競業避止義務の範囲が広い条項

5. **弁護士・行政書士への確認事項をリストアップする。** 以下の場合は必ず専門家確認を促す：
   - 損害賠償条項に重大なリスクがある
   - 知的財産権の帰属が不明確
   - 労働法規制に関わる条件がある
   - 金額が大きい・長期間の契約

6. **整理メモを作成する。** 以下の形式でまとめる：

   ```
   契約書整理メモ

   【基本情報】
   - 契約種別: {種類}
   - 契約期間: {期間}
   - 自動更新: あり/なし（{条件}）

   【重要条項サマリー】
   ・報酬: {金額と条件}
   ・解除条件: {条件}
   ・秘密保持: {期間と対象}

   【注意すべき論点】
   🔴 重要: {リスクの説明}
   🟡 要確認: {確認事項}

   【弁護士・行政書士への確認推奨事項】
   1. {確認事項}
   2. {確認事項}

   ※ このメモは法的アドバイスではありません。
     最終的な契約締結の判断は弁護士・行政書士にご確認ください。
   ```

## Approval gates

- **法的に問題ないとの確定的な判断は下さない。**
- **「このまま署名できる」「法的に問題ありません」とは言わない。**
- **オーナーの明示的な承認なしに、契約書の署名・送信・電子契約（クラウドサイン・GMOサイン等）の実行は行わない。**
- **弁護士・行政書士への相談を促す。**
